# textAnimationsOnVideo
Text Animations on Video

## Installation instructions
- clone this repo
- create a virtualenv and activate it
- when inside the textAnimationsOnVideo dir use 'pip3 install -r requirements.txt' to install the requirements

## Usage instructions
- python3 manage.py runserver

- Open a browser and point it to: http://localhost:8000/app/1/animations/   to see the json output
- Open a browser and point it to: http://localhost:8000/app/1/ to see the animations on the video

- To add animations go the the admin panel: http://localhost:8000/admin/app/animation/ and add a new animation or remove existing animation
- Then go to the video's page or the json output to see the changes
